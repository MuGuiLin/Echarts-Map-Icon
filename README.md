# Echarts-Map-Icon

> 基于 Echarts.js 的 Map 地图，在上面添加自定义标图（图片），并自动循环提示相关信息！
>
> Echarts Map 地图，自定义标图（图片），自动循环显示，ECharts 如何自定义省、市、县区地图(各省市直辖区的 map 地图)，ECharts 如何在地图上自定义图片、图标，根据经纬度显示自定义建筑图片、添加自定义覆盖物。
