# Echarts-Map-Icon


> 基于Echarts.js 的 Map地图，在上面添加自定义标图（图片），并自动循环提示相关信息！
>
> Echarts Map地图，自定义标图（图片），自动循环显示，ECharts如何自定义省、市、县区地图(各省市直辖区的map地图)，ECharts如何在地图上自定义图片、图标，根据经纬度显示自定义建筑图片、添加自定义覆盖物。
